## ❌ 常见错误处理

### 错误 1: 端口被占用

```
OSError: [Errno 48] Address already in use
```

**解决**:
```bash
# 查找占用端口的进程
lsof -i :8000
lsof -i :5000

# 杀死进程
kill -9 <PID>
```

### 错误 2: 连接被拒绝

```
ConnectionRefusedError: [Errno 61] Connection refused
```

**解决**: 检查对应的服务器是否已启动

### 错误 3: 模块导入错误

```
ModuleNotFoundError: No module named 'fastapi'
```

**解决**:
```bash
pip install -r requirements_api.txt
```

## 🎯 测试 API 接口

### 测试注册接口

```bash
curl -X POST http://localhost:8000/api/register_app \
  -H "Content-Type: application/json" \
  -d '{
    "app_name": "test_app",
    "callback_url": "http://localhost:5000",
    "capabilities": [
      {
        "name": "test_action",
        "type": "navigation",
        "description": "测试动作",
        "params": []
      }
    ]
  }'
```

### 查看已注册的 APP

```bash
curl http://localhost:8000/api/apps
```

### 创建任务

```bash
curl -X POST http://localhost:8000/api/task \
  -H "Content-Type: application/json" \
  -d '{
    "message": "测试任务",
    "enable_interaction": false
  }'
```

### 查询任务状态

```bash
curl http://localhost:8000/api/task/{task_id}
```

## 📊 完整流程图

```
客户端测试 (client_example.py client)
    │
    ├─1. 注册 APP
    │   POST http://localhost:8000/api/register_app
    │   {
    │     "app_name": "douyin",
    │     "callback_url": "http://localhost:5000",
    │     "capabilities": [...]
    │   }
    │   
    │   ↓
    │   
    │   Agent 系统 (api_server.py)
    │   创建 RemoteAppHandler
    │   注册到 AppRegistry
    │   
    ├─2. 创建任务
    │   POST http://localhost:8000/api/task
    │   {"message": "找AI视频"}
    │   
    │   ↓
    │   
    │   Agent 系统
    │   ├─ GlobalIntentAI: 识别意图
    │   ├─ ContextBuilderAI: 构建上下文
    │   ├─ StrategyAI: 生成策略
    │   └─ TaskRuntime: 开始执行
    │   
    ├─3. Agent 决策并回调
    │   
    │   DecisionAI 决定: 执行 "next"
    │   ↓
    │   RemoteAppHandler.execute("next", [])
    │   ↓
    │   POST http://localhost:5000/execute
    │   {"action": "next", "params": []}
    │   
    │   ↓
    │   
    │   调用方服务器 (client_example.py server)
    │   执行实际操作
    │   返回: {"success": true, "data": {...}}
    │   
    │   ↓
    │   
    │   Agent 系统
    │   PerceptionAI: 分析结果
    │   DecisionAI: 继续决策...
    │   
    └─4. 查询任务状态
        GET http://localhost:8000/api/task/{task_id}
        返回: {"status": "completed", ...}
```

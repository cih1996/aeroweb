## 🔄 交互流程详解

### 步骤 1: 注册 APP 能力

```python
# 调用方 → Agent 系统
POST http://agent-server:8000/api/register_app
{
  "app_name": "douyin",
  "callback_url": "http://your-server:5000",
  "capabilities": [
    {"name": "next", "type": "navigation", ...},
    {"name": "getVideoInfo", "type": "read", ...},
    {"name": "like", "type": "engage", ...}
  ]
}

# Agent 系统内部
→ 创建 RemoteAppHandler("douyin", "http://your-server:5000", ...)
→ 注册到 AppRegistry
```

### 步骤 2: 创建任务

```python
# 调用方 → Agent 系统
POST http://agent-server:8000/api/task
{
  "message": "帮我找5个AI视频",
  "enable_interaction": false
}

# Agent 系统内部（后台执行）
→ GlobalIntentAI: 识别意图 {"app": "douyin", "goal": "找AI视频"}
→ ContextBuilderAI: 构建上下文 {"domain": "短视频", ...}
→ StrategyAI: 生成策略 {"min_match_score": 0.6, ...}
→ TaskRuntime: 初始化状态机
→ 开始主循环...
```

### 步骤 3: 决策并回调执行

```python
# Agent 系统内部
→ SupervisorAI: 风险检查 {"control": "CONTINUE"}
→ DecisionAI: 决定下一步 {"action": "next", "params": []}
→ ExecutorAI: 生成执行计划

# Agent 系统 → 调用方
POST http://your-server:5000/execute
{
  "action": "next",
  "params": []
}

# 调用方执行
→ 你的代码：切换到下一个视频
→ 返回结果：{"success": true, "data": {...}}

# Agent 系统接收结果
→ PerceptionAI: 判断内容 {"match_score": 0.88, "should_proceed": true}
→ DecisionAI: 继续决策 {"action": "like", ...}
→ 循环...
```

### 步骤 4: 任务完成

```python
# Agent 系统内部
→ 达到目标（找到5个视频）
→ SummarizerAI: 总结任务
→ MemoryLearnerAI: 学习记忆
→ 保存到 memory_storage/

# 调用方查询状态
GET http://agent-server:8000/api/task/{task_id}
→ {"status": "completed", "progress": {...}, "token_stats": {...}}
```

## 🧠 8 层 AI 详解

### L1: GlobalIntentAI
- **作用**: 理解用户意图
- **输入**: 用户消息
- **输出**: `{"app": "douyin", "goal": "找AI视频", "keywords": ["AI", "人工智能"]}`
- **记忆**: 读取 L0 用户画像

### L2: ContextBuilderAI
- **作用**: 构建任务上下文
- **输入**: 用户意图
- **输出**: `{"domain": "短视频", "risk_level": "low", ...}`
- **记忆**: 读取 L0/L1/L2

### L3: StrategyAI
- **作用**: 生成执行策略
- **输入**: 任务上下文 + APP 能力
- **输出**: `{"min_match_score": 0.6, "preferred_engagement_action": "like", ...}`
- **记忆**: 读取 L1/L2 策略模板

### L4: DecisionAI
- **作用**: 动态决策下一步
- **输入**: 当前状态 + 策略
- **输出**: `{"action": "next", "params": [], "reason": "..."}`
- **记忆**: 读取 L2 决策偏好 + 会话历史

### L5: ExecutorAI
- **作用**: 生成执行计划
- **输入**: 决策指令
- **输出**: `{"action": "next", "validation": "ok"}`
- **记忆**: 禁止读取（纯函数）

### L6: PerceptionAI
- **作用**: 感知判断内容
- **输入**: 对象信息 + 用户意图
- **输出**: `{"match_score": 0.88, "should_proceed": true, ...}`
- **记忆**: 禁止读取（纯函数）

### L7: SupervisorAI
- **作用**: 风险控制
- **输入**: 运行状态 + 执行指标
- **输出**: `{"control": "CONTINUE"/"SLEEP"/"STOP", ...}`
- **记忆**: 读取 L0/L1/L2 用户偏好

### L8: MemoryLearnerAI
- **作用**: 学习并生成记忆
- **输入**: 任务总结 + 运行统计
- **输出**: `{"memories": [...]}`
- **记忆**: 读取全部，写入全部

## 💾 记忆系统

### 三层架构

```
L0 (Global)
  ├─ USER_PROFILE        # 用户画像
  └─ 永久保存

L1 (App)
  ├─ USER_PROFILE        # APP 级偏好
  └─ TTL: 180 天

L2 (Intent)
  ├─ DECISION_PREFERENCE # 决策偏好
  ├─ STRATEGY_TEMPLATE   # 策略模板
  └─ TTL: 60 天
```

### 四级成熟度

1. **OBSERVATION** (观测) - 新记忆
2. **CANDIDATE** (候选) - 出现 2+ 次
3. **STABLE** (稳定) - confidence > 0.7
4. **RULE** (规则) - confidence > 0.9

### 自动演化

```python
# 第一次任务
MemoryLearner 生成: OBSERVATION (confidence=0.6)

# 第二次相同场景
系统检测到相似记忆
→ 合并 → 升级为 CANDIDATE (confidence=0.7)

# 多次确认后
→ 升级为 STABLE (confidence=0.85)

# 用户明确调整
→ 升级为 RULE (confidence=0.95)
```

## 🔒 安全建议

1. **HTTPS**
   - 生产环境必须使用 HTTPS
   - 保护 API Key 和数据传输

2. **API Key 验证**
   ```python
   # 在 api_server.py 中添加
   from fastapi import Header
   
   @app.post("/api/task")
   async def create_task(
       request: TaskRequest,
       x_api_key: str = Header(None)
   ):
       if x_api_key != "your-secret-key":
           raise HTTPException(401, "Unauthorized")
   ```

3. **IP 白名单**
   ```python
   ALLOWED_IPS = ["192.168.1.100", "10.0.0.1"]
   
   @app.middleware("http")
   async def check_ip(request, call_next):
       if request.client.host not in ALLOWED_IPS:
           return JSONResponse({"error": "Forbidden"}, 403)
       return await call_next(request)
   ```

4. **限流保护**
   ```python
   from slowapi import Limiter
   
   limiter = Limiter(key_func=get_remote_address)
   
   @app.post("/api/task")
   @limiter.limit("10/minute")
   async def create_task(...):
       ...
   ```

# 测试图片目录

这个目录用于存放文件上传劫持功能使用的测试图片。

## 使用方法

1. 将测试图片文件放在此目录下
2. 支持的图片格式：`.jpg`, `.jpeg`, `.png`
3. 建议的文件名：
   - `test1.jpg`
   - `test2.png`
   - `test3.jpg`

## 工作原理

当用户在浏览器中选择文件上传时，系统会自动拦截并替换为这个目录中的测试图片。

系统会按以下顺序查找可用的测试图片：
1. `test1.jpg`
2. `test2.png`
3. `test3.jpg`

如果找不到任何测试图片，系统会使用一个默认的占位图片。

## 注意事项

- 图片文件需要放在此目录下才能被系统识别
- 建议使用较小的图片文件（< 5MB）以提升性能
- 图片会被转换为 base64 格式注入到页面中

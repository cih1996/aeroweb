AI 专属浏览器

为 AI Agent 设计的浏览器，通过 CLI/MCP 控制网页操作。 支持多缓存实例隔离，让 AI 像人一样浏览网页。

快速开始 → ⭐ GitHub
Terminal
$ aeroweb tab new "https://github.com"
✓ 标签页已创建 (tab_1)

$ aeroweb tab screenshot @last -o screenshot.png
✓ 截图已保存

$ aeroweb tab exec @last -e "document.title"
"GitHub: Let's build from here"

核心特性

专为 AI Agent 设计的浏览器能力

⚡

CLI 控制

通过命令行完全控制浏览器，打开页面、点击、输入、截图、执行 JS，一切皆可自动化。

🔒

多缓存隔离

每个会话独立缓存，支持多账号同时登录同一网站，互不干扰。

🔄

智能复用

相同网站自动复用已有标签页，避免重复创建，节省资源。

📸

截图 & 快照

一键截图或获取 DOM 快照，让 AI 能"看到"页面内容。

🖱️

模拟交互

点击、输入、上传文件，模拟真实用户操作，绕过反爬检测。

🔌

MCP 协议

支持 Model Context Protocol，与 Claude 等 AI 无缝集成。

一键安装

选择你的操作系统，复制命令即可安装

curl -fsSL https://github.com/cih1996/aeroweb/releases/latest/download/aeroweb-macos-$(uname -m | sed 's/x86_64/x64/;s/aarch64/arm64/') -o /tmp/aeroweb && chmod +x /tmp/aeroweb && /tmp/aeroweb
curl -fsSL https://github.com/cih1996/aeroweb/releases/latest/download/aeroweb-linux-x64 -o /tmp/aeroweb && chmod +x /tmp/aeroweb && /tmp/aeroweb
curl -fsSL https://github.com/cih1996/aeroweb/releases/latest/download/aeroweb-win-x64.exe -o aeroweb.exe && aeroweb.exe
